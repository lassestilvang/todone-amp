DATABASE SCHEMA
================================================================================

Templates Table:
- id (primary)
- name, description (content)
- category, isPrebuilt (metadata)
- data (template structure: sections, tasks)
- usageCount (tracking)
- ownerId (ownership)
- createdAt, updatedAt (timestamps)

Indices:
- [category+isPrebuilt] - Browse by category
- [ownerId+createdAt] - User's templates

UserTemplates Table:
- id (primary)
- userId, templateId (relationships)
- isFavorite (user preference)
- lastUsedAt (tracking)
- createdAt (timestamp)

Indices:
- [userId+templateId] - Lookup user template
- [userId+isFavorite] - Favorite queries
